# Unity_2D_Platformer_Final-Project
This is a 2D action-platformer developed in Unity as the final submission for the "Introduction to Game Development" course. The project serves as a practical application of the fundamental techniques and workflows taught throughout the semester.

Key Features
Dual-Level Design: Includes two distinct levels with increasing difficulty and platforming challenges.

Combat Mechanics: Integrated enemy AI with two specific attack patterns:

Melee: Enemies equipped with swords for close-quarters combat.

Ranged: Enemies utilizing arrows for distance-based engagement.

UI System: * Main Menu to start the experience.

Functional Pause Menu for real-time game management.

Course Techniques: Developed using standard industry practices learned in class, including Sprite management, Prefabs, C# Scripting, and Unity's Physics 2D system.
